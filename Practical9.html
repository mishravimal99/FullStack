Introduction to HTML

HTML stands for Hyper Text Markup Language. HTML is the combination of Hypertext and Markup language. Markup language is used to define the text document within tag which defines the structure of web pages. This language is used to annotate text so that a machine can understand it

Introduction to CSS

Cascading Style Sheets, fondly referred to as CSS, is a simply designed language intended to simplify the process of making page presentable. CSS allows you to apply styles to web pages. More importantly, CSS enables you to do this independent of the HTML that makes up each web page.

Introduction to JS

JavaScript is a lightweight, cross-platform and interpreted scripting language. It is well-known for the development of web pages, many non-browser environments also use it. JavaScript can be used for Client-side developments as well as Server-side developments. It contains a lot of useful libraries
